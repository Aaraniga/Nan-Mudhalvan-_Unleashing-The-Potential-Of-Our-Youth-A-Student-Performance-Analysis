  • 02 what is story in data analysis

    A data story typically involves presenting the data in a meaningful way, using visualizations, charts, and graphs to convey key messages and trends. It goes beyond just presenting raw data and focuses on explaining the significance and implications of the analysis.

  • 03 What does a report contain in Data analysis?

    A report in data analysis typically contains a comprehensive analysis of the data and presents the findings in a structured and organized manner. It serves as a means to communicate the results of the data analysis process to stakeholders, decision-makers, or clients. The specific contents of a data analysis report may vary depending on the objectives, audience, and context of the analysis.
